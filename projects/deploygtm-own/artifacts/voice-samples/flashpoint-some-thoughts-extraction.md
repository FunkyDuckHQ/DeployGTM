## What This Sample Shows

This deck is a strong example of Matthew's strategic voice when writing to an opportunity after live conversation and manual refinement. It is concise, humble, commercially direct, and anchored in what can be tested rather than what can be claimed.

## Voice Traits

- Humble but not passive: the deck repeatedly names uncertainty while still making a clear recommendation.
- Diagnostic first: it starts from existing revenue, repeat behavior, proof, and current motion before proposing new activity.
- Plainspoken operating logic: the language is easy to follow and avoids big-market performance language.
- Math-literate: the revenue gap is translated into concrete motion options instead of left as an abstract target.
- Bet-oriented: the recommendation is framed as something to test over 30 to 60 days.
- Founder-aware: it preserves enterprise work while suggesting a parallel agency lane, instead of forcing a false binary.
- Recurrence-aware: it separates repeat services from true recurring value and looks for what happens between projects.

## Reusable Strategic Moves

- Start from current revenue as evidence, not from a desired GTM theory.
- Ask what repeats before adding more motion.
- Identify the proof point that can carry the story.
- Compare motions using operating math: volume, project weight, repeatability, founder time, and proof creation.
- Treat the first 30 days as action and the first 60 days as learning.
- Convert a business thesis into a specific execution sequence by week.

## Reusable Language Patterns

- "I don't know the whole business yet. My read..."
- "There is enough signal to test a real motion."
- "What repeats, what should become recurring, and where I'd start selling."
- "Before adding more motion..."
- "The current revenue is a map."
- "The question is what keeps them using [product]."
- "The first 60 days should tell us which path is real."
- "By day 60, we should know what to double down on."

## Deck Shape

1. Cover: personal note, caveat, and thesis.
2. Diagnosis: what to solve first from existing revenue.
3. Wedge: where to start selling without abandoning current high-value work.
4. Revenue math: model the gap and compare the paths.
5. Recurring revenue: distinguish project repetition from durable recurring value.
6. Execution: first 30 days with 60-day learning criteria.
