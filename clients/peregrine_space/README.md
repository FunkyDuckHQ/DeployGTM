# Peregrine Space Client Workspace

This is the Phase 1 demo client workspace for DeployGTM.

## Layout

- `config/`: scoring, workflow, vendor, and signal definitions.
- `inputs/`: account and source input data.
- `outputs/`: generated score snapshots and reports.
- `runs/`: execution logs keyed by execution ID.

No real vendor integrations are enabled in Phase 1.
